# APP 端

## 快速开发

```bash
npm i
npm run dev
```

`dev` 会同时启动 `dev:react` 和 `dev:tcm`

- `dev:react` 启动 react 热更新进程
- `dev:tcm` 启动 tcm 自动生成 css 的 ts 描述文件 \*.css.d.ts
